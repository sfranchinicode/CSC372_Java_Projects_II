# Bank Balance GUI Application

## Overview
This project is a Java Swing GUI application that allows a user to manage a simple bank account. Users can deposit money, withdraw money, view their balance, and exit the program while displaying the final balance.

The program uses the BankAccount class from Module 1 for all account logic.

## Project Structure
BankAccount.java (from Module 1)  
BankBalanceGUI.java  

## Features

- Deposit funds into the account
- Withdraw funds from the account
- Display current balance
- Show final balance on exit
- Simple graphical interface using Java Swing

### Classes

#### BankAccount.java (Module 1)
Handles all account data and banking logic.

**Key Methods:**
- deposit()
- withdraw()
- getBalance()
- setters and getters

#### BankBalanceGUI.java
Handles all user interaction through a graphical interface.

**GUI Components:**
- JPanel
- JButton
- JLabel
- JTextField

**Buttons:**
- Deposit
- Withdraw
- Show Balance
- Exit

## Concepts Demonstrated
- Object-Oriented Programming (OOP)
- Encapsulation
- Event-driven programming
- Java Swing GUI development
- Reusing existing classes (Module 1 integration)
